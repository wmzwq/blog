Git

Git常用命令

$ git init
$ git remote add origin <url>
$ git status
$ git add .
$ git commit -m "描述"
$ git push -u origin master
$ git push --force --set-upstream origin master  git本地仓库强制替换远程
$ git branch 分支名   创建本地分支
$ git checkout 分支名 切换本地分支
